### Prerequisites

* Node.js (LTS version recommended)
* npm (Node Package Manager) or Yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/cryptofoxtech/cryptofox-corp-website.git](https://github.com/cryptofoxtech/cryptofox-corp-website.git)
    ```
2.  **Navigate into the project directory:**
    ```bash
    cd cryptofox-corp-website
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or if you use Yarn:
    # yarn install
    ```
